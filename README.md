# ibimal/homebrew-tap

Homebrew tap for [KanataBar](https://github.com/ibimal/kanatabar) — a macOS
supervisor and menu-bar app for the [kanata](https://github.com/jtroo/kanata)
keyboard remapper.

## Install

```sh
brew install --cask ibimal/tap/kanatabar
```

`brew upgrade` is the only update mechanism; the app never self-updates.
